# Create React App

- $ yarn create react-app react-create-app-intro
- Built a project (folder)
- $ cd react-create-app-intro
- node_modules folder
- src (source) - App.js - main component
- public - index.html
- $ yarn start
- Terminal is directing your computer to act as a server
- control + c - ends the server process
- Deleted React boilerplate from App.js

```
import React from 'react'

class App extends React.Component {
  render() {
    return(
      <>
        <h1>Hi Bravo!!!</h1>
      </>
    )
  }
}

export default App
```
